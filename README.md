# 🏦 Bank Management System

A simple and efficient **Bank Management System** developed in **C++** that allows users to perform basic banking operations such as account creation, deposits, withdrawals, balance inquiries, and account management through a console-based interface.

---

## 📌 Features

* Create a new bank account
* View account details
* Deposit money
* Withdraw money
* Check account balance
* Delete account records
* Store and retrieve data using file handling
* Menu-driven user interface

---

## 🛠️ Technologies Used

* **C++**
* Object-Oriented Programming (OOP)
* File Handling
* Standard Template Library (STL)

## 🚀 Getting Started

### Prerequisites

* C++ Compiler (GCC, MinGW, or MSVC)
* Any C++ IDE such as Code::Blocks, VS Code, Dev-C++, or Visual Studio

### Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/Bank-Management-System.git
```

2. Navigate to the project directory:

```bash
cd Bank-Management-System
```

3. Compile the source code:

```bash
g++ BankManagementSystemBankManagementSystem.cpp -o bank
```

4. Run the executable:

```bash
./bank
```
